# 🎵 RhythmVerse

A modern, feature-rich music streaming web application built with vanilla JavaScript, HTML, and CSS. RhythmVerse provides a beautiful interface for discovering and playing music from tour own Music API, complete with personalized recommendations, listening history, and an immersive player experience.

## ✨ Features

### 🎧 Core Features
- **Full Music Player** - Play, pause, seek, shuffle, loop, and repeat functionality
- **Smart Playback Control** - Keyboard shortcuts (Space to play/pause, Arrow keys for navigation)
- **Quality Selection** - Stream at 320kbps, 160kbps, 96kbps, or 48kbps
- **Persistent Sessions** - Remembers your current song, position, volume, and quality settings
- **Offline Retry** - Automatically retries failed requests when connection is restored

### 🧠 Smart Recommendation Engine
- **Knowledge Graph** - Curated artist relationships, languages, eras, and moods
- **Taste Learning** - Builds a profile from your searches and listening behavior
- **Time-of-Day Awareness** - Adapts recommendations based on when you listen
- **Artist Co-occurrence** - Learns which artists you tend to play together
- **Completion Tracking** - Differentiates between skipped and fully-played songs
- **Multiple Rail Types** - Artist-based, era-based, mood-based, and term-based suggestions

### 📊 Features
- **Listening History** - Track every song you play with timestamps and completion status
- **Search** - Live search with typeahead suggestions from your history and taste profile
- **Smart Suggestions** - Shows recent searches, artist suggestions, and learned search terms
- **Shareable Links** - Direct links to any song with artist parameters

### 🎨 UI/UX
- **Dual Player Modes** - Mini player (always visible) and maximized full-screen player
- **Ambient Effects** - Dynamic orbs, responsive animations, and visual feedback
- **Visual Progress** - Seek flash animations (10s forward/back) with arrow indicators
- **Loading States** - Animated MD3-style spinners with error states
- **Responsive** - Optimized for both desktop and mobile devices
